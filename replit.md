# DevTools Hub

## Overview

A developer-focused web platform combining **free, browser-only dev tools** (JSON formatter, base64, password generator, UUID, color converter, text-case converter, Markdown preview, JWT decoder, URL encoder, hash generator) with a **SEO-optimized tech blog** for programming-error articles. Includes contact form, newsletter signup, dark/light mode, and AdSense-ready ad slots.

## Stack

- **Monorepo tool**: pnpm workspaces, Node 24, TypeScript 5.9
- **Web app** (`artifacts/devtools-blog`): React + Vite, wouter (routing), TanStack Query, shadcn/ui, Tailwind, lucide-react, marked + DOMPurify + highlight.js (markdown), crypto-js (hashing)
- **API server** (`artifacts/api-server`): Express 5, Drizzle ORM
- **Database**: PostgreSQL + Drizzle (chosen over the user-suggested MongoDB so the app fits the platform's first-class Postgres support — communicated to the user)
- **API contract**: OpenAPI YAML at `lib/api-spec/openapi.yaml`, Zod schemas + typed React-Query hooks generated by Orval

## Key Commands

- `pnpm run typecheck` — typecheck everything
- `pnpm --filter @workspace/api-spec run codegen` — regenerate Zod + React-Query hooks
- `pnpm --filter @workspace/db run push` — push DB schema (dev)
- `pnpm --filter @workspace/scripts run seed` — reseed authors / categories / tags / posts

## Architecture notes

- **API routes**: posts (list/featured/recent/popular/by-slug/related), categories, tags, tools, stats, contact, newsletter — under `artifacts/api-server/src/routes/`.
- **Tools list**: static array in `artifacts/api-server/src/routes/tools.ts`. Each tool's UI lives client-side in `artifacts/devtools-blog/src/components/tools/<slug>.tsx`, dispatched by `pages/tool-detail.tsx`.
- **Pagination**: API uses `offset` + `limit`. The web app converts `(page-1)*limit` to offset.
- **SEO**: `useSEO()` hook updates `<title>` + meta description + og tags per page.
- **Markdown rendering**: `src/lib/markdown.ts` parses with marked, syntax-highlights with highlight.js, sanitizes with DOMPurify.
